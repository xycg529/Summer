# 逆康托展开
## 思路
假设求4位数中第19个位置的数字。

19减去1  → 18

18 对3！作除法 → 得3余0

0对2！作除法 → 得0余0

0对1！作除法 → 得0余0

据上面的可知：

我们第一位数（最左面的数），比第一位数小的数有3个，显然 第一位数为→ 4

比第二位数小的数字有0个，所以 第二位数为→1

比第三位数小的数字有0个，因为1已经用过，所以第三位数为→2

第四位数剩下3

该数字为4123
## 代码实现
```c
//逆康托展开函数 
// BY WTU 计科11902 xycg529
int fact(int n)//定义阶乘函数 
{
	int i, sum = 1;
	for(i = 1; i <= n; i++){
		sum = sum*i;
	}
	return sum;
}
int  fac[] = {1,1,2,6,24,120,720,5040,40320};
//康托展开的逆运算,{1...n}的全排列，中的第k个数为s[]
void recantor(int n, int k, char s[])
{
    int i, j, t, a[8] = {0};
    --k;
    for( i = 0; i < n; i++)
    {
        t = k / fact[n-i-1];
        for( j = 1; j <= n; j++)
            if (!a[j]){
                if (t == 0){
                	break;
				}
                --t;
            }
        s[i] = '0'+j;
        a[j] = 1;
        k %= fact[n-i-1];
    }
}
```
